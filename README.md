# LLM Evaluation Toolkit

This repository contains tools for evaluating Large Language Models (LLMs)
using automated testing pipelines and human feedback scoring.

## Features

- Prompt testing framework
- Model response evaluation
- Hallucination detection experiments
- Benchmark scoring metrics

## Technologies

Python  
PyTorch  
HuggingFace Transformers  
Jupyter Notebook
